# Задачи

**№1**  
Получите список пользователей и их транзакций.

**№2**  
Получите список идентификаторов транзакций и имена пользователей, которые их сделали, для первых 5 транзакций.

**№3**  
Получите имена пользователей и идентификатор транзакции, которые оплатили покупки картой (CARD).

**№4**  
Получите имена пользователей и идентификатор транзакции, которые оплатили покупки кэшем (CASH).

**№5**  
Получите имя того пользователя, у которого сумма покупок наибольшая среди всех остальных.
