## Задания

**№1.** Найдите минимальное значение колонки `amount` из этой таблицы.

**№2.** Найдите максимальное значение колонки `amount` из этой таблицы.

**№3.** Найдите среднее значение колонки `amount` из этой таблицы.

**№4.** Найдите среднее значение колонки `amount` из этой таблицы и округлите это значение.

**№5.** Найдите сумму колонки `amount` из этой таблицы и округлите это значение до одного знака после точки.

**№6.** Найдите среднее значение колонки `amount` из этой таблицы и округлите это значение в большую сторону.

**№7.** Найдите среднее значение колонки `amount` из этой таблицы и округлите это значение в меньшую сторону.

**№8.** Найдите сумму колонки `amount` из этой таблицы и сделайте так, чтобы осталось 4 знака после точки.
