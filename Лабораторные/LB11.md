## Задачи

Составьте SQL-запросы для решения следующих задач.

### Задача №1
Получите список пользователей и их транзакций. Выведите также имя магазина, где была совершена данная транзакция.

**Ожидаемые столбцы в результате:**  
`user_id`, `name` (имя пользователя), `id` (идентификатор транзакции), `amount`, `payment_type`, `created_at`, `store_name` (название магазина).

---

### Задача №2
Получите список идентификаторов транзакций и имена пользователей, которые их сделали, для первых 5 транзакций. Также выведите имя магазина, где была совершена данная транзакция.

**Ожидаемые столбцы в результате:**  
`id` (идентификатор транзакции), `name` (имя пользователя), `store_name` (название магазина).

---

### Задача №3
Получите имена пользователей и идентификатор транзакции, которые оплатили покупки картой. Также выведите имя магазина, где была совершена данная транзакция.

**Ожидаемые столбцы в результате:**  
`name` (имя пользователя), `id` (идентификатор транзакции), `store_name` (название магазина).

---

### Задача №4
Получите имена пользователей и идентификатор транзакции, которые оплатили покупки наличными. Также выведите имя магазина, где была совершена данная транзакция.

**Ожидаемые столбцы в результате:**  
`name` (имя пользователя), `id` (идентификатор транзакции), `store_name` (название магазина).

## Указания к выполнению

1. Используйте операторы `JOIN` для соединения таблиц по соответствующим ключам.
2. Для фильтрации по типу оплаты используйте оператор `WHERE`.
3. Для ограничения количества строк используйте оператор `LIMIT` (где требуется).
4. Запросы должны быть написаны в стандартном SQL (поддерживаемом в PostgreSQL/MySQL).
