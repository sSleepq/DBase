## Задания

**Задание 1. Подсчет активности клиентов**

Напишите SQL-запрос, который выводит имя каждого клиента и общее количество сделанных им заказов. Результат должен быть отсортирован по имени клиента в алфавитном порядке.

*Требования к решению:*
- Использовать агрегатную функцию `COUNT()`.
- Использовать оператор `GROUP BY`.
- Использовать оператор `ORDER BY`.

**Задание 2. Поиск неактивных клиентов**

Напишите SQL-запрос, который находит клиентов, не совершивших ни одного заказа. Выведите идентификатор и имя клиента.

*Требования к решению:*
- Использовать тип соединения `LEFT JOIN` или `RIGHT JOIN`.
- Использовать условие `WHERE ... IS NULL` для фильтрации результатов.
