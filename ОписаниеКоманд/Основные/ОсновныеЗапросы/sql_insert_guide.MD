# Команда `INSERT`


Команда `INSERT` вставляет записи в базу данных. Можно добавить как одну запись, так и сразу несколько.


## Синтаксис

**Первый синтаксис:**
```sql
INSERT INTO имя_таблицы SET поле1=значение1, поле2=значение2, поле3=значение3...
```

**Второй синтаксис:**
```sql
INSERT INTO имя_таблицы (поле1, поле2...) VALUES (значение1, значение2...)
```

**Добавление сразу нескольких записей:**
```sql
INSERT INTO имя_таблицы (поле1, поле2...)
VALUES (значение1, значение2...), (значение1, значение2...)...
```

## Таблицы для примеров


**Таблица `employees`:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|------------|--------------|------------------|
| 1         | user1      | 23           | 400              |
| 2         | user2      | 25           | 500              |

## Примеры использования

### Пример 1: добавление записи первым синтаксисом

Добавим нового работника (user3) с возрастом 23 и зарплатой 500, используя первый синтаксис:

```sql
INSERT INTO employees SET name='user3', age=23, salary=500;
```

**Таблица `employees` после выполнения команды:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|------------|--------------|------------------|
| 1         | user1      | 23           | 400              |
| 2         | user2      | 25           | 500              |
| 3         | user3      | 23           | 500              |

### Пример 2: добавление записи вторым синтаксисом


Добавим того же работника (user3) с возрастом 23 и зарплатой 500, используя второй синтаксис:

```sql
INSERT INTO employees (name, age, salary) VALUES ('user3', 23, 500);
```

**Таблица после выполнения команды (аналогична предыдущему результату):**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|------------|--------------|------------------|
| 1         | user1      | 23           | 400              |
| 2         | user2      | 25           | 500              |
| 3         | user3      | 23           | 500              |

### Пример 3: добавление нескольких записей

Добавим одновременно двух работников:
- user3 (возраст 23, зарплата 500);
- user4 (возраст 30, зарплата 1000).


```sql
INSERT INTO employees (name, age, salary) VALUES ('user3', 23, 500), ('user4', 30, 1000);
```

**Таблица после выполнения команды:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|------------|--------------|------------------|
| 1         | user1      | 23           | 400              |
| 2         | user2      | 25           | 500              |
| 3         | user3      | 23           | 500              |
| 4         | user4      | 30           | 1000             |

> **Примечание:** в исходном примере указана зарплата 900 для user4, но по команде вставляется 1000. В таблице приведён корректный результат согласно SQL‑запросу.

### Пример 4: добавление записи без указания одного из полей


Добавим работника user3 с возрастом 23, не указывая зарплату (будет использовано значение по умолчанию):


```sql
INSERT INTO employees (name, age) VALUES ('user3', 23);
```

**Таблица после выполнения команды:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|------------|--------------|------------------|
| 1         | user1      | 23           | 400              |
| 2         | user2      | 25           | 500              |
| 3         | user3      | 23           | 0                |

> **Примечание:** значение зарплаты по умолчанию — 0.
