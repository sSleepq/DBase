# Команда `DELETE`

Команда `DELETE` удаляет записи из базы данных. Записи для удаления задаются с помощью команды `WHERE`.

**Важно:** команда `WHERE` не является обязательной. Если её не указать, будут удалены все записи в таблице. Будьте внимательны — так случайно можно уничтожить все данные.

## Синтаксис

```
DELETE FROM имя_таблицы WHERE условие
```

## Таблицы для примеров

**Таблица `employees`:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|------------|--------------|------------------|
| 1         | user1      | 23           | 400              |
| 2         | user2      | 25           | 500              |
| 3         | user3      | 23           | 500              |

## Примеры использования

### Пример 1: удаление пользователя по `id`

Удалим пользователя с `id`, равным 2:

```sql
DELETE FROM employees WHERE id = 2;
```

**Таблица `employees` после выполнения команды:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|------------|--------------|------------------|
| 1         | user1      | 23           | 400              |
| 3         | user3      | 23           | 500              |

### Пример 2: удаление пользователей по условию

Удалим пользователей с `id`, большим 1:

```sql
DELETE FROM employees WHERE id > 1;
```

**Таблица `employees` после выполнения команды:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|------------|--------------|------------------|
| 1         | user1      | 23           | 400              |
