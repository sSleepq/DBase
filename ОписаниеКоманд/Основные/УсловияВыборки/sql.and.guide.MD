# Команда AND в SQL


## Описание

Команда **AND** — это логический оператор «И», который позволяет комбинировать условия в предложении `WHERE`.


Оператор `AND` возвращает значение `TRUE` **только если все** указанные условия истинны. Если хотя бы одно условие ложно, запись не попадает в результат.


## Синтаксис

```sql
WHERE условие1 AND условие2 [AND условие3 ... AND условиеN]
```

## Пример таблицы

Рассмотрим примеры на таблице `employees`:


| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 400 |
| 2 | user2 | 25 | 500 |
| 3 | user3 | 23 | 500 |
| 4 | user4 | 30 | 900 |
| 5 | user5 | 27 | 500 |
| 6 | user6 | 28 | 900 |

## Примеры использования

### Пример 1: выбор по диапазону id

**Задача:** выбрать записи, у которых `id` больше или равен 2 **и** меньше или равен 4 (оба условия должны выполняться одновременно).

**Запрос:**
```sql
SELECT * FROM employees WHERE id >= 2 AND id <= 4;
```

**Результат:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 2 | user2 | 25 | 500 |
| 3 | user3 | 23 | 500 |
| 4 | user4 | 30 | 900 |

*Пояснение:*  
- Запись с `id = 2` подходит: `2 >= 2` (истинно) **и** `2 <= 4` (истинно).  
- Запись с `id = 3` подходит: `3 >= 2` (истинно) **и** `3 <= 4` (истинно).  
- Запись с `id = 4` подходит: `4 >= 2` (истинно) **и** `4 <= 4` (истинно).

### Пример 2: выбор по возрасту и зарплате

**Задача:** выбрать записи, у которых возраст равен 23 **и** зарплата равна 400 (оба условия обязательны).

**Запрос:**
```sql
SELECT * FROM employees WHERE age = 23 AND salary = 400;
```

**Результат:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 400 |

*Пояснение:*  
- Запись с `id = 1` подходит: `age = 23` (истинно) **и** `salary = 400` (истинно).  
- Запись с `id = 3` не подходит: хотя `age = 23` (истинно), но `salary = 500` (ложно).


## Важные замечания

1. Оператор `AND` требует выполнения **всех** перечисленных условий одновременно.
2. При комбинировании с другими операторами (например, `OR`) рекомендуется использовать скобки `()` для явного указания приоритета условий.
3. Условия могут быть сложными (подзапросы, функции), но каждое должно возвращать булево значение (`TRUE`, `FALSE` или `UNKNOWN`).
4. `AND` «сжимает» результирующий набор: чем больше условий через `AND`, тем меньше строк в результате (если условия не избыточны).
5. В отличие от `OR`, который «расширяет» результат (достаточно одного истинного условия), `AND` строго фильтрует данные.
