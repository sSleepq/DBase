# Команда OR в SQL


## Описание

Команда **OR** — это логический оператор «ИЛИ», который позволяет комбинировать условия в предложении `WHERE`.


Оператор `OR` возвращает значение `TRUE`, если **хотя бы одно** из условий истинно.


## Синтаксис

```sql
WHERE условие1 OR условие2 [OR условие3 ... OR условиеN]
```

## Пример таблицы

Рассмотрим примеры на таблице `employees`:


| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 400 |
| 2 | user2 | 25 | 500 |
| 3 | user3 | 23 | 500 |
| 4 | user4 | 30 | 900 |
| 5 | user5 | 27 | 500 |
| 6 | user6 | 28 | 900 |


## Примеры использования

### Пример 1: выбор по возрасту или зарплате

**Задача:** выбрать записи, у которых зарплата равна 400 **или** возраст равен 23 (достаточно выполнения хотя бы одного условия).

**Запрос:**
```sql
SELECT * FROM employees WHERE age = 23 OR salary = 400;
```

**Результат:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 400 |
| 3 | user3 | 23 | 500 |


*Пояснение:*  
- Запись с `id = 1` подходит, потому что `age = 23` **и** `salary = 400` (оба условия истинны).  
- Запись с `id = 3` подходит, потому что `age = 23` (первое условие истинно, второе — ложно).

### Пример 2: выбор по идентификатору

**Задача:** выбрать записи, у которых `id` равен 2 **или** 3.

**Запрос:**
```sql
SELECT * FROM employees WHERE id = 2 OR id = 3;
```

**Результат:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 2 | user2 | 25 | 500 |
| 3 | user3 | 23 | 500 |

*Пояснение:*  
- Запись с `id = 2` подходит, потому что первое условие истинно.  
- Запись с `id = 3` подходит, потому что второе условие истинно.

## Важные замечания

1. Оператор `OR` проверяет, истинно ли **хотя бы одно** из перечисленных условий.
2. Если в запросе используется несколько логических операторов (например, `AND` и `OR`), то порядок выполнения можно изменить с помощью скобок `()`.
3. Условия могут быть сколь угодно сложными (включая подзапросы и функции), но каждое из них должно возвращать булево значение (`TRUE`, `FALSE` или `UNKNOWN`).
