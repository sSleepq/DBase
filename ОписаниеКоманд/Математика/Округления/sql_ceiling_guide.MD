# Функция CEILING в SQL

## Описание

Функция **CEILING** используется для округления дробных чисел до целого **в большую сторону** (т. е. к ближайшему большему целому числу).

Примеры работы функции:
- `CEILING(100.1)` → 101
- `CEILING(200.4)` → 201
- `CEILING(-1.2)` → -1


## Синтаксис

```sql
SELECT CEILING(поле) FROM имя_таблицы WHERE условие;
```

## Пример таблицы

Таблица `employees`:

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 100.1 |
| 2 | user2 | 24 | 200.4 |
| 3 | user3 | 25 | 300.9 |

## Пример использования

**Задача:** округлить зарплату до целого числа в большую сторону.

**SQL‑запрос:**
```sql
SELECT *, CEILING(salary) AS ceiling_salary FROM employees;
```

**Результат:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) | ceiling_salary (округлённая зарплата) |
|-----------|-----------|-------------|--------------|-------------------------------|
| 1 | user1 | 23 | 100.1 | 101 |
| 2 | user2 | 24 | 200.4 | 201 |
| 3 | user3 | 25 | 300.9 | 301 |

## Важные замечания

1. **CEILING всегда округляет вверх**, независимо от дробной части:
   - даже если дробная часть очень мала (например, 100.1), результат будет 101;
   - для отрицательных чисел: `CEILING(-1.2) = -1` (так как -1 больше -1.2).


2. Функция работает только с **числовыми типами данных** (DECIMAL, FLOAT, DOUBLE и т. п.).

3. В отличие от `ROUND`, который округляет по математическим правилам, `CEILING` всегда «поднимает» число до следующего целого вверх.


4. Результат `CEILING` — целое число (тип INT или аналогичный), без дробной части.


5. Функция `CEILING` является аналогом математической функции «потолок» (⌈x⌉).
