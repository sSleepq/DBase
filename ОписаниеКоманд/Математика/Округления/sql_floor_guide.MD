# Функция FLOOR в SQL

## Описание

Функция **FLOOR** используется для округления дробных чисел до целого **в меньшую сторону** (т. е. к ближайшему меньшему целому числу).

Например:
- `FLOOR(100.1)` → 100
- `FLOOR(200.9)` → 200
- `FLOOR(-1.2)` → -2

## Синтаксис

```sql
SELECT FLOOR(поле) FROM имя_таблицы WHERE условие;
```

## Пример таблицы

Таблица `employees`:

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 100.1 |
| 2 | user2 | 24 | 200.4 |
| 3 | user3 | 25 | 300.9 |

## Пример использования

**Задача:** округлить зарплату до целого числа в меньшую сторону.


**SQL‑запрос:**
```sql
SELECT *, FLOOR(salary) AS floor_salary FROM employees;
```

**Результат:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) | floor_salary (округлённая зарплата) |
|-----------|-----------|-------------|--------------|-------------------------------|
| 1 | user1 | 23 | 100.1 | 100 |
| 2 | user2 | 24 | 200.4 | 200 |
| 3 | user3 | 25 | 300.9 | 300 |

> **Примечание:** в исходном примере для `user3` указано значение `301`, но это ошибка — `FLOOR(300.9)` равно `300`, так как функция всегда округляет **вниз** до ближайшего целого.


## Важные замечания

1. **FLOOR всегда округляет вниз**, независимо от дробной части:
   - даже если дробная часть близка к 1 (например, 300.9), результат будет 300;
   - для отрицательных чисел: `FLOOR(-1.2) = -2` (так как -2 меньше -1.2).

2. Функция работает только с **числовыми типами данных** (DECIMAL, FLOAT, DOUBLE и т. п.).

3. В отличие от `ROUND`, который округляет по математическим правилам, `FLOOR` всегда «отсекает» дробную часть, оставляя только целую часть числа, уменьшенную до ближайшего целого вниз.


4. Результат `FLOOR` — целое число (тип INT или аналогичный), без дробной части.
