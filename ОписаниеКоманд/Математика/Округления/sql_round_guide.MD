# Функция ROUND в SQL

## Описание

Функция **ROUND** используется для округления дробных чисел до нужного знака в дробной части.

## Синтаксис

- **Округление до целого числа:**
  ```sql
  SELECT ROUND(поле) FROM имя_таблицы WHERE условие;
  ```

- **Округление до определённого количества знаков после запятой:**
  ```sql
  SELECT ROUND(поле, сколько_знаков_оставить) FROM имя_таблицы WHERE условие;
  ```

## Пример таблицы

Таблица `employees`:

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 100.129 |
| 2 | user2 | 24 | 200.391 |
| 3 | user3 | 25 | 300.934 |

## Примеры использования

### Пример 1. Округление до целых чисел

**Задача:** округлить зарплату до целых чисел при выборке.


**SQL‑запрос:**
```sql
SELECT *, ROUND(salary) AS salary FROM employees;
```

**Результат:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 100 |
| 2 | user2 | 24 | 200 |
| 3 | user3 | 25 | 301 |

### Пример 2. Округление до двух знаков после точки


**Задача:** округлить зарплату до двух знаков после запятой.


**SQL‑запрос:**
```sql
SELECT *, ROUND(salary, 2) AS salary FROM employees;
```

**Результат:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 100.13 |
| 2 | user2 | 24 | 200.39 |
| 3 | user3 | 25 | 300.93 |

### Пример 3. Округление до одного знака после точки

**Задача:** округлить зарплату до одного знака после запятой.

**SQL‑запрос:**
```sql
SELECT *, ROUND(salary, 1) AS salary FROM employees;
```

**Результат:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 100.1 |
| 2 | user2 | 24 | 200.4 |
| 3 | user3 | 25 | 300.9 |

### Пример 4. Округление до целых с указанием параметра 0

**Задача:** округлить зарплату до целых, явно указав количество знаков после запятой (0).

**SQL‑запрос:**
```sql
SELECT *, ROUND(salary, 0) AS salary FROM employees;
```

**Результат:**

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 100 |
| 2 | user2 | 24 | 200 |
| 3 | user3 | 25 | 301 |

## Важные замечания

1. Если второй параметр (`сколько_знаков_оставить`) не указан, число округляется до целого.
2. При указании `0` в качестве второго параметра результат также будет целым числом (без дробной части).
3. Функция `ROUND` следует стандартным правилам математического округления:
   - если цифра после заданного знака ≥ 5, предыдущий знак увеличивается на 1;
   - если < 5 — остаётся без изменений.
4. `ROUND` работает с числовыми типами данных (DECIMAL, FLOAT, DOUBLE и др.).
