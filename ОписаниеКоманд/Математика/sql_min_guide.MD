# Функция MIN в SQL

## Описание

Функция **MIN** возвращает минимальное значение поля среди найденных строк.

## Синтаксис

```sql
SELECT MIN(поле) FROM имя_таблицы WHERE условие;
```

## Пример таблицы

Таблица `employees`:

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 400 |
| 2 | user2 | 25 | 500 |
| 3 | user3 | 23 | 500 |
| 4 | user4 | 30 | 900 |
| 5 | user5 | 27 | 500 |
| 6 | user6 | 28 | 900 |

## Примеры использования

### Пример 1. Нахождение минимального значения зарплаты

**Задача:** найти минимальное значение зарплаты в таблице.

**SQL‑запрос:**
```sql
SELECT MIN(salary) AS min FROM employees;
```

**Результат:**

| min (минимальная зарплата) |
|------------------------|
| 300 |

### Пример 2. Нахождение минимальной зарплаты с условием

**Задача:** найти минимальное значение зарплаты среди записей, у которых `id` больше или равен 2.

**SQL‑запрос:**
```sql
SELECT MIN(salary) AS min FROM employees WHERE id >= 2;
```

**Результат:**

| min (минимальная зарплата) |
|------------------------|
| 400 |

### Пример 3. Нахождение минимального и максимального значений одновременно

**Задача:** получить максимальное и минимальное значение зарплаты одновременно.

**SQL‑запрос:**
```sql
SELECT MIN(salary) AS min, MAX(salary) AS max FROM employees;
```

**Результат:**

| min (минимальная зарплата) | max (максимальная зарплата) |
|------------------------|---------------------------|
| 300 | 700 |
