# Функция AVG в SQL

## Описание

Функция **AVG** возвращает среднее арифметическое значение указанного поля среди найденных строк. 

Среднее арифметическое вычисляется как сумма всех значений, делённая на их количество:  
$$
\text{AVG} = \frac{\sum \text{значений}}{\text{количество значений}}
$$

Например, если в таблице хранятся возраста пользователей, функция `AVG` позволяет найти их средний возраст.

## Синтаксис

```sql
SELECT AVG(поле) FROM имя_таблицы WHERE условие;
```

## Пример таблицы

Таблица `employees`:

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 100 |
| 2 | user2 | 25 | 200 |
| 3 | user3 | 23 | 300 |

## Пример использования

### Нахождение средней зарплаты

**Задача:** вычислить среднюю зарплату по всем сотрудникам в таблице.

**SQL‑запрос:**
```sql
SELECT AVG(salary) AS avg FROM employees;
```

**Результат:**

| avg (средняя зарплата) |
|--------------------|
| 200 |

**Пояснение:**  
- Сумма зарплат: $100 + 200 + 300 = 600$  
- Количество записей: 3  
- Среднее значение: $600 / 3 = 200$
