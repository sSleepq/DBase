# Функция MAX в SQL

## Описание

Функция **MAX** возвращает максимальное значение поля среди найденных строк.

## Синтаксис

```sql
SELECT MAX(поле) FROM имя_таблицы WHERE условие;
```

## Пример таблицы

Таблица `employees`:

| id (айди) | name (имя) | age (возраст) | salary (зарплата) |
|-----------|-----------|-------------|--------------|
| 1 | user1 | 23 | 400 |
| 2 | user2 | 25 | 500 |
| 3 | user3 | 23 | 500 |
| 4 | user4 | 30 | 900 |
| 5 | user5 | 27 | 500 |
| 6 | user6 | 28 | 900 |

## Примеры использования

### Пример 1. Нахождение максимального значения зарплаты

**Задача:** найти максимальное значение зарплаты в таблице.

**SQL‑запрос:**
```sql
SELECT MAX(salary) AS max FROM employees;
```

**Результат:**

| max (максимальная зарплата) |
|--------------------------|
| 900 |

> **Примечание:** в исходном примере указан результат «700», но по данным таблицы максимальное значение зарплаты — 900.

### Пример 2. Нахождение максимальной зарплаты с условием

**Задача:** найти максимальное значение зарплаты среди записей, у которых `id` меньше или равен 4.

**SQL‑запрос:**
```sql
SELECT MAX(salary) AS max FROM employees WHERE id <= 4;
```

**Результат:**

| max (максимальная зарплата) |
|--------------------------|
| 900 |

> **Примечание:** в исходном примере указан результат «600» и заголовок «min», что является ошибкой. По данным таблицы для `id <= 4` максимальная зарплата — 900.

### Пример 3. Нахождение минимального и максимального значений одновременно

**Задача:** получить максимальное и минимальное значение зарплаты одновременно.

**SQL‑запрос:**
```sql
SELECT MAX(salary) AS max, MIN(salary) AS min FROM employees;
```

**Результат:**

| max (максимальная зарплата) | min (минимальная зарплата) |
|--------------------------|---------------------------|
| 900 | 400 |

> **Примечание:** в исходном примере указаны значения «700» и «300», что не соответствует данным таблицы. Корректные значения: максимум — 900, минимум — 400.
